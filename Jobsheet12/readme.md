## Pertanyaan
1. SLL vs DLL: Single Linked List punya 1 pointer (next), hanya bisa maju. Double Linked List punya 2 pointer (next, prev), bisa maju dan mundur .
2. Fungsi pointer: next untuk lanjut ke data depan; prev untuk mundur ke data belakang. Keduanya mempermudah navigasi dan manipulasi tanpa looping dari awal.
3. Fungsi Konstruktor: Mengatur head dan tail menjadi null, yang menandakan bahwa linked list awalnya sepenuhnya kosong.
4. head = tail = newNode: Karena jika list kosong, node baru yang masuk adalah satu-satunya elemen (menjadi yang pertama sekaligus yang terakhir).
