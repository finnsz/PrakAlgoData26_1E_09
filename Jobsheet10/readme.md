## Pertanyaan
1. Kenapa -1 dan 0? size = 0 karena antrian masih kosong. front dan rear = -1 sebagai penanda belum menunjuk indeks array mana pun.
2. Kode rear di Enqueue: Untuk memutar penunjuk rear kembali ke indeks 0 jika sudah mencapai batas akhir array (Circular Queue).
3. Kode front di Dequeue: Memutar penunjuk front kembali ke indeks 0 jika data di ujung akhir array dikeluarkan.
4. Kenapa i = front? Karena data antrian terdepan tidak selalu diam di indeks 0. Kita harus mulai mencetak dari posisi front saat ini.
5. Maksud i = (i + 1) % max: Rumus untuk bergeser indeks secara melingkar. Jika sudah mencapai batas maksimum (max), indeks otomatis kembali ke 0.
6. Kode Overflow: Blok if (IsFull()) di dalam method Enqueue.
